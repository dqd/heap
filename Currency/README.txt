Another Supybot Currency plugin, this time based on Fixer.io/Alpha Vantage API.

If you want to use Alpha Vantage API, which has more capabilities, you need to:

1. Install the alpha_vantage Python module:
https://github.com/RomelTorres/alpha_vantage

2. Obtain an API key from https://www.alphavantage.co/support/ and set it up
like that:
/msg <yourbot> config plugins.Currency.api_key <APIKEY>
